Duxravel

基于 Laravel 驱动的开源后台管理开发框架

快速开始    获取源码
composer create-project duxphp/duxravel

产品特点

Duxravel 解决了中小型 Web 项目的开发效率与体验的痛点

开箱即用

1分钟快速安装，安装后即可进入开发无需复杂配置。

模块化开发

App 式开发架构低耦合高内聚，多人协作互不干涉。

命令行脚手架

简单几句命令搞定重复工作，可轻松建立初始应用模块。

微前端架构

集成 Vue 单页实时编译与 JSON DOM编译器，免去二次打包。

Arco UI 库

集成开源的 Arco 前端组件，优化用户操作体验与美观的设计。

多后台架构

多后台公共设计，可轻松开发多用途后台功能。

应用化扩展

通过应用打包可轻松将应用分享给他人或出售使用。

简单权限

基于路由的多角色权限设计，无需在数据库进行复杂权限配置。

菜单接入

接口化菜单接入，可将应用轻松接入任意菜单模块中，无需后台配置。

Api 鉴权

集成基础 Api 开发能力，可直接上手进行接口开发系统集成鉴权机制。

Laravel PHP

基于 Laravel 作为基础框架开发框架让开发者得心应手。

100% 开源

遵循标准的 MIT 开源协议，让开发产品无后顾之忧。
